# Running vagrant

* Commands
  * `vagrant up`<br/>
    Start vm if not started and provision.
  * `vagrant ssh`<br/>
    Log into the box.
  * `vagrant destroy`<br/>
    Halt and remove vm.
  * `vagrant provision`<br/>
    Provision a vm that is already running.
  * `vagrant halt`<br/>
    Shutdown a vm without destroying: saves state.
  * `vagrant status`<br/>
    Show the status of virtual machines.
